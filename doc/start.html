简介：LAU（lying-admin-ui）是一款基于layui的后台前端框架，使用LAU的基本前提是熟悉layui的模块化设计和使用，如果不懂，请先查看 layui模块化用法
初衷：被 layuiAdmin 深深的吸引了，想要使用，无奈的是刚结婚的自己手头紧巴巴的，于是就想动手做一个一样的后台框架（这可能也是我编码的动力吧）
构造： layui的后台布局 + layui + layuiAdmin的部分css ，有些样式是自己写不来的，抄袭了layuiAdmin，希望贤心谅解
交流： 码外社区 、 QQ群 、 github
开始使用
├─doc           //文档页面目录
├─html          //示例页面目录
├─static        //静态资源目录
│  │─css        //LAU的css目录
│  │─data       //示例页面数据目录
│  │─js         //LAU的js目录
│  └─layui      //layui项目(当前版本修复了https://github.com/sentsin/layui/pull/121，后续贤心修复的话会跟进，尽量不修改layui的源代码)
├─.gitignore    //LAU忽略文件
│─favicon.ico   //网站图标
│─index.html    //LAU主页
└─README.md     //LAU说明文件
